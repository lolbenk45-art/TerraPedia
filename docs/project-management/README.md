# Project Management

本目录保存当前仍需追溯的项目管理材料。这里记录项目状态、决策、风险、迁移记录和协调日志，不承接 `project-plan/` 的项目级源头职责。

## 当前入口

- `current-status.md`：当前状态。
- `decision-log.md`：决策记录。
- `risk-register.md`：风险登记。
- `daily/`：每日项目记录。
- `frontend-migration/`：前端迁移计划、批次摘要和相关工具，原路径为 `front/migration/**`。

## 落位规则

- 项目级状态、决策、风险、迁移记录和协调日志放在这里。
- 项目级长期计划、SOP、项目总览和历史计划索引放 `project-plan/`。
- 任务级执行计划放 `docs/plans/`。
- 已完成或过期的项目计划放 `project-plan/archive/`。
- 临时过程材料先放 `task/`，不要直接写入本目录。
