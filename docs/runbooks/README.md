# 运行手册

本目录保存可重复执行的运行、启动、排障和维护手册。运行手册描述稳定操作流程，不保存一次性执行日志或项目计划。

## 适合放这里

- 本地启动排障手册。
- 数据导入或回滚操作手册。
- 线上或本地运维流程。

## 不适合放这里

- 一次性任务日志。
- 未确认的草稿。
- 大批量脚本输出。
- 项目级长期计划和历史计划，应放 `project-plan/`。
- 任务级执行计划，应放 `docs/plans/`。

- `docs/runbooks/image-asset-cache.md`：图片 source/cache/fallback 缓存手册。
