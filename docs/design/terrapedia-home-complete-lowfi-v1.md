# TerraPedia 完整首页低保真 v1

日期：2026-05-16

文件：`docs/design/terrapedia-home-complete-lowfi-v1.svg`

## 组成

这份完整首页由三段已确认设计合成：

1. 首屏 Hero：`terrapedia-home-lowfi-v2-5.svg`
   - 左侧图鉴入口主视觉
   - 右侧标题、搜索和快捷入口

2. 首页下半部分：`terrapedia-home-lower-lowfi-v6.svg`
   - 探索地图
   - 精选路线
   - Boss 事件条
   - 图鉴卷轴
   - 独立索引台

3. 底部 Footer：`footer-v1-camp-exit`
   - 品牌牌匾
   - 资料索引
   - 数据来源/版权/路径提示

## 设计原则

- 首页负责第一印象和探索动机，允许比内页更有舞台感。
- 下半部分从视觉入口过渡到资料探索，不做普通卡片堆叠。
- Footer 是旅程收束，不是链接垃圾桶。
- 色彩控制在深绿、苔绿、暖金、纸色，避免多色脏感。
