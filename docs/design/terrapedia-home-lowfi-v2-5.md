# TerraPedia 首页低保真 v2.5

日期：2026-05-16

文件：`docs/design/terrapedia-home-lowfi-v2-5.svg`

## 定稿方向

首页采用“游戏图鉴型”首屏，但控制在可落地的强度内：

- 左侧是主视觉：巨型图鉴入口、柔和光晕、少量环绕内容卡。
- 右侧是主任务：品牌标题、说明文案、全站搜索、快捷分类入口。
- 底部是数据规模浮层：保留可信度和内容体量信号，但不抢主视觉。
- 背景是 Terraria 世界感：像素地形、远山、暖金/苔绿/矿物青氛围。

## 页面结构

1. 顶部导航
   - Logo / brand mark
   - 公开内容入口
   - 主题或辅助入口

2. 首屏主体
   - 左侧：Codex / 图鉴入口视觉
   - 右侧：H1、简短说明、主搜索框
   - 快捷入口：物品、NPC、Boss、文章/指南

3. 首屏底部
   - 内容规模统计
   - 露出下一屏，提示页面可继续下滑

## 高保真建议

- 主视觉后续应替换为真实像素风图层或精修插画，不建议只靠 CSS 块面。
- 搜索框应是首页首屏的最高优先级交互。
- 快捷入口可以使用真实游戏图标或稀有度色条增强辨识度。
- 首页可以比内页更戏剧化；列表页和详情页仍应回到内容密度和可扫描性。
