# TerraPedia 辅助页面低保真 v1

日期：2026-05-16

分支：`design/public-pages-lowfi-2026-05-16`

文件：`docs/design/terrapedia-utility-pages-lowfi-v1.svg`

## 覆盖页面

### 1. 关于页

适用路由：

- `/about`

设计意图：

- 延续首页的世界观入口，但降低冲击强度。
- 首屏用深绿舞台、世界地图台和路径节点建立项目可信度。
- 下方数据链路改成流程地图，不做普通说明条。
- 项目边界、贡献入口、数据来源使用三张带纹理的资料牌，保持“资料站档案感”。

### 2. 认证页组

适用路由：

- `/login`
- `/register`
- `/forgot-password`

设计意图：

- 登录、注册、找回密码共用同一个视觉框架，避免三套散乱页面。
- 左侧保留 Terraria 世界入口感，并加入传送门式登录意象。
- 右侧用纸质表单承载输入，外层叠加符文圆环，弱化普通表单感。
- 底部三个小状态块表示同模板的三种模式切换。

### 3. 个人中心

适用路由：

- `/profile`

设计意图：

- 顶部是玩家档案柜牌匾，展示头像、身份和贡献统计。
- 左侧为账户/收藏/投稿/设置导航。
- 主区承载最近收藏、投稿草稿、账户安全等模块，并使用档案柜纹理提升识别度。
- 底部增加账户状态路径，避免变成普通个人资料页。

### 4. 写作与文案工作台

适用路由：

- `/articles/write`
- `/copywriting`

设计意图：

- 使用同一套三栏工作台：素材/模板、正文编辑、右侧建议与发布设置。
- 视觉上更像资料工坊，而不是普通 Markdown 编辑器。
- 顶部工具条加入符文纹理，中央稿纸加入内框和分隔线，左侧素材区更像资料夹。
- 右侧深色工具栏承载 AI 文案、关联图鉴、发布检查等辅助能力。

### 5. 404 页面

适用路由：

- `/:pathMatch(.*)*`

设计意图：

- 404 不做普通空状态，改成迷失地图和破损图鉴。
- 提供回首页和继续搜索两个主动作。
- 增加迷失路径、地图节点和底部推荐路径，让错误页也有探索导向。
- 和首页视觉有联系，但氛围更安静，适合作为错误页。

## 视觉约束

- 继续使用深绿、苔绿、暖金、纸色。
- 辅助页比首页克制，不抢首页第一印象。
- 认证页和 404 可使用深色世界背景；个人中心和工作台优先纸色信息密度。
- 避免新增高饱和多色块，防止页面显脏。
- 所有页面都保留统一导航骨架，便于之后抽成组件。
