# Archived Project Plans

本目录保存已完成、已废弃或只做历史参考的项目计划。归档内容是历史证据，不作为当前执行入口。

归档规则：

- 先在 `project-plan/INDEX.md` 标注状态。
- 再移动文件。
- 如果文件仍被文档引用，先更新引用或保留兼容说明。
- 历史正文可能保留旧路径、旧根目录布局或当时的绝对路径；除非影响当前索引或当前执行入口，不批量改写旧路径。
- `project-plan/plan-/**` 是独立的历史 legacy cluster，不从那里批量搬入 archive。

## 当前分区

- `2026-03/`：2026-03 的历史迁移、数据库和数据源计划。
- `2026-04/`：2026-04 的里程碑计划、执行记录、验收记录和阶段问题汇总。
- `v1/`：早期 V1.0 决策和项目计划。

历史归档正文可能保留当时的绝对路径或旧根目录引用。除非影响当前入口索引，不批量改写历史记录。
