## 关键问题与处理

### 1. Cloudflare 拦截页面直连

现象：

- `requests.get(https://terraria.wiki.gg/zh/wiki/微光)` 返回 `403 Blocked`

处理：

- 改为只使用官方 MediaWiki API
- Python 通过 `requests + BeautifulSoup` 解析 `action=parse&prop=text`

### 2. 微光页面不是单一实体

页面同时包含：

1. 物品嬗变
2. 拆解特例
3. 小动物和敌怪嬗变
4. NPC 转化

处理：

- 统一按这四类拆分提取
- 后续管理端和数据库也按四张表承接

### 3. 初版解析未收口

解析过程中曾出现：

- 误把表头当成数据导入
- 标题解析走大类跳转页
- 中文标题与现有实体表不一致

处理：

- 修正表格数据行识别逻辑
- 缩小候选标题范围
- 使用 DB `items/npcs` 双语字段参与解析
- 引入 redirect 回填
- 补充最后一批显式 alias 映射

最终结果：

- `unresolvedTitles = 0`
- `wiki_shimmer_manifest.parseStatus = parsed`
