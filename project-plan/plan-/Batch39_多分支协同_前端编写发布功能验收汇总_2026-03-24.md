## 3. 前端功能实现（已完成）

### 3.1 变更文件
- `data-query-app/pages/articles.vue`

### 3.2 功能行为
- 文章编辑弹窗新增双提交意图：
  - `Save Draft`
  - `Publish Now`
- 逻辑保持富文本编辑器与实时预览不变。
- 新建文章点击 `Publish Now`：
  - 先创建草稿
  - 再调用 `updateStatus(id, 'PUBLISHED')` 完成立即发布
- 编辑已有文章点击 `Publish Now`：
  - 先保存更新
  - 再调用 `updateStatus(id, 'PUBLISHED')`
- 评审流表格动作（Submit Review / Approve / Reject / Publish / Offline）保持不变。
- 防重复提交：
  - `submitting` 状态期间直接返回，按钮禁用。
- 交互文案更清晰：
  - 区分 `Saving draft...` 与 `Publishing...`

### 3.3 关键实现点
- `pages/articles.vue`
  - `submitForm(intent: 'draft' | 'publish')`
  - 发布意图下在保存后串行调用 `articlesStore.updateStatus(savedArticle.id, 'PUBLISHED')`
- `stores/articles.ts`
  - 为 `createArticle / updateArticle / updateStatus` 增加可选参数 `{ silent?: boolean }`
  - 发布串行流程下抑制中间提示，避免重复 toast 噪音。
