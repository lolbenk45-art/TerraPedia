# Batch37 文章接入前端汇总（2026-03-23）

## 目标
将后端文章审核机制接入前端管理端，支持完整流程：
- 新建/编辑文章
- 提交审核
- 审核通过/驳回
- 发布/下线
- 审核日志查看

## 本次改动

### 1. 管理端文章 Store 接口扩展
文件：`data-query-app/stores/articles.ts`

新增类型与字段：
- `ArticleReviewStatus`: `DRAFT | PENDING_REVIEW | APPROVED | REJECTED`
- `ArticleReviewAction`: `APPROVE | REJECT`
- `ArticleReviewLog`
- `AdminArticle` 增加：`reviewStatus/reviewComment/reviewedAt/submittedAt/reviewerName`

新增接口方法：
- `fetchArticleById(id)`
- `submitReview(id)` -> `POST /admin/articles/{id}/submit-review`
- `reviewArticle(id, action, comment?)` -> `POST /admin/articles/{id}/review`
- `publishArticle(id)` -> `POST /admin/articles/{id}/publish`
- `offlineArticle(id)` -> `POST /admin/articles/{id}/offline`
- `fetchReviewLogs(id, page, size)` -> `GET /admin/articles/{id}/review-logs`

保留兼容方法：
- `updateStatus(id, nextStatus)`（旧状态接口）

### 2. 管理端文章页面重构
文件：`data-query-app/pages/articles.vue`

完成内容：
- 修复原页面乱码与破损结构，重建文章管理页。
- 新增审核流动作按钮（按状态动态显示）：
  - `DRAFT/REJECTED`：提交审核
  - `PENDING_REVIEW`：通过 / 驳回
  - `APPROVED`：发布
  - `PUBLISHED`：下线
- 新增“驳回原因”弹窗（驳回必须填写原因）。
- 新增“审核日志”弹窗与分页。
- 保留文章新建/编辑能力（新建默认 DRAFT，编辑保留当前文章状态）。
- 表格新增审核相关信息展示：`reviewStatus/reviewer/submittedAt/reviewedAt/reviewComment`。
