# 本地执行指令（2026-03-18）

本文档用于在当前仓库状态下快速复现本地开发链路与缓存验证结果。

## 一键启动

在仓库根目录执行：

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\scripts\dev\start-local-stack.ps1
```

停止：

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\scripts\dev\stop-local-stack.ps1
```

关键约定（与脚本一致）：

- 管理端使用 `pnpm run dev -- --host localhost --port 3001` 启动。
- 后端通过 `run-backend-v1-local.ps1` 启动，并显式注入：
  - `SPRING_PROFILES_ACTIVE=v1-local`
  - `TERRAPEDIA_REDIS_HOST=127.0.0.1`
  - `TERRAPEDIA_REDIS_PORT=6380`
  - `TERRAPEDIA_REDIS_PASSWORD=root`

## 启动后访问

- 前台：`http://localhost:5174`
- 管理端：`http://localhost:3001`
- 后端 Swagger：`http://localhost:8888/api/swagger-ui.html`

## Redis 验证

当前本地 Redis：

- 地址：`127.0.0.1:6380`
- 密码：`root`

基础验证步骤：

```text
AUTH root
PING
SET codex:test ok
GET codex:test
```

预期：

- `AUTH root` -> `OK`
- `PING` -> `PONG`
- `SET` -> `OK`
- `GET` -> `ok`

## 缓存验证

### 统计缓存

```text

```

```text

```
