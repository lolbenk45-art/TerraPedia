- 将文章内容主字段从 `content_markdown` 迁移为 `content_html`（前后端联动，兼容旧字段）。

  - `back/src/main/resources/mapper/ArticleMapper.xml`

  - `back/src/main/resources/db/migration/V11__rename_article_content_column_to_html.sql`

  - `scripts/dev/start-local-stack.ps1`
- 启动结果：

  - Admin: `http://localhost:3001/`
  - Back API Docs: `http://localhost:8888/api/v3/api-docs`

### 3.2 启动故障与修复
- 问题1：前端/管理端曾出现 `esbuild spawn EPERM`
  - 根因：在受限执行环境中 Node 子进程创建受限。
  - 处理：通过已授权的非沙箱启动脚本拉起，前台与管理端恢复正常。
- 问题2：后端 Flyway 校验失败（`V7` checksum mismatch）
  - 报错：`Migration checksum mismatch for migration version 7`
  - 处理：对本地库执行一次等价 `flyway repair` 的 checksum 修复后重启。
  - 结果：后端成功启动，且 `V11` 迁移执行成功。
